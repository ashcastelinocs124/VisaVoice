# visavoice

visavoice is a voice-based AI receptionist for UIUC International Student and Scholar Services (ISSS). It answers inbound calls over a LiveKit/Twilio telephony stack, handles FAQ lookups against ISSS documentation, books advising appointments, and escalates complex or sensitive cases to human staff. See the design doc at `docs/plans/2026-04-19-visavoice-design.md` for the full architecture.
